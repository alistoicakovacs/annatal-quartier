# docs/

Project documentation and non-runtime files. **None of this is required for the
live website.** The site itself is, at the repo root: `index.html`, `styles.css`,
`script.js`, `datenschutz.html`, `impressum.html`, `logos/`, and the form backend
`sendmail.php`.

| Folder | Contents |
|--------|----------|
| **briefing/** | `Wissensdatei …pdf` — the client briefing / source of truth (git-ignored). |
| **planning/** | `PLAN.md` (implementation plan), `DESIGN-MODERNIZATION.md` (design notes). |
| **styleguide/** | Living style guide — open `styleguide.html` in a browser. Renders against the real `../../styles.css` + `logos/`. (`styleguide.css`, `STYLEGUIDE.md`.) |
| **superpowers/** | Brainstorming `plans/` and `specs/` from the redesign. |

> Note: `CLAUDE.md` stays at the repo root by convention (Claude Code loads it from there). The styleguide files are currently untracked in git.
